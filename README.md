## Requirement
Python3  
numpy  
scipy  
pandas  
matplotlib

## Usage
```
import ffbutil as fb
```
You can find some example in sample folder.  

## Install
システム管理者の場合
```
pip3 install git+https://github.com/YuyaM/ffbutil
```
ユーザーの場合
```
pip3 install --user git+https://github.com/YuyaM/ffbutil
```
